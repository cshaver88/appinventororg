2.

Sketch the blocks for making a ball move down at a speed of 30 pixels a second. The ball should move smoothly so specify an appropriate timer interval (you donâ€™t need to show the blocks, just note what you will set the timer interval to in the designer). Show your math.


With the above blocks, set the Clock2.TimerInterval to 50. Explanation: If you didn't care that the ball moved smoothly, you'd set the TimerInterval to 1000 (a second) and you'd move by 30 pixels each time. But you don't want choppy movement, so you need a smaller Clock.TimerInterval, say 50 ms. You know the Clock.Timer event will then be triggered every 50 ms. The question then is how far to move the ball each 50 ms in order to get a speed of 30 pixels/second. The equation to setup is this: x/50=30/1000, where x is the amount to move. If you cross-multiply, you get x*1000=30*50, or x=1.5.
