1.

(100) Consider a music app that plays do-re-mi over and over. The music should start playing when the user clicks a "PlayButton", and continue playing until the user clicks a "StopButton". It should always start with "do.mp3" and there should be a 2 second delay between the start of each note. You can use the following blocks:

Answer.

The Clock.Timer event is triggered every Clock.TimerInterval. In order to delay two second between notes, set the Clock.TimerInterval to 2000 (2000 milliseconds= 2 seconds).
