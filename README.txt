appinventor.org directory information


The purpose of this readme is to explain what goes where in the app inventor project directory

Pages:
	All html files 
		-Related should be organized together into subfolders e.g. html files used in the course system go in the
		course system folder inside of the Pages folder.
	
Assets
	-Basicly everything else, put it into the corresponding subfolder, and again try to group related items into
		meta subfolders
